Directions

Working with Rasters

Complete the assignment Rmd file in the data folder. Follow the tutorial from Andrew MacLachlan, but don’t start from scratch, start from the Project folder/file here: https://www.dropbox.com/s/wpg4aqcqrdsesrh/lab9.zip?dl=0

To download data from Earth Explorer:

  1. On the Search tab, once you zoom in appropriately to your location, click the Use Map button, then click Data Sets to advance to the next section.
  2. Select Landsat > Landsat Collection 2 Level-1 > Landsat 8-9 OLI/TRS C2 L1, then click Results. You might get a warning that you can ignore.
  3. Click on the footprint button for the appropriate coverage. Once you find the appropriate image, click the download icon.
  4. In Download Options dialog box that appears, click the Product Options button and then click the top button for the “Bundle” - you’ll then need to unzip the files (no subfolders) into the lab9/data/landsat folder.
